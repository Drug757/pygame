# Техническое задание

## 1. Описание задачи
Нужно сделать простую 2D-игру с помощью Pygame, где игрок управляет квадратом и должен избегать столкновения с препятствием. Игра работает в реальном времени и реагирует на нажатия клавиш.

## 2. Цель работы
Создать простое игровое приложение с ограничением FPS, чтобы движение было плавным и одинаковым на разных компьютерах.

## 3. Функциональные требования
Программа должна:
- открывать игровое окно  
- отображать игрока и препятствие  
- позволять двигаться с клавиатуры (WASD)  
- проверять столкновения  
- завершать игру при столкновении  
- работать с фиксированной частотой кадров (60 FPS)  

## 4. Нефункциональные требования
- игра должна работать стабильно  
- движение должно быть плавным  
- не должно быть слишком быстрой или слишком медленной реакции  

## 5. Ограничения и допущения
- только 2D-графика  
- без звуков и сложных эффектов  
- один уровень  
- всё реализовано в одном файле  
- используется фиксированный FPS (60 кадров в секунду)

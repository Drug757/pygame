# Pygame 

## 1. Краткое описание инструмента
Pygame — это библиотека для Python, с помощью которой можно создавать простые 2D-игры и интерактивные приложения. Она позволяет работать с графикой, клавиатурой и событиями, не вдаваясь в сложные детали.

## 2. Цель изучения
Целью было понять основы разработки игр: как работает игровой цикл, как обрабатываются нажатия клавиш и как реализуется движение объектов на экране.

## 3. Описание реализованной задачи
В рамках работы была реализована простая игра, где игрок управляет квадратом и должен избегать столкновения с препятствием. При столкновении игра завершается. Также было добавлено ограничение в 60 FPS, чтобы движение было стабильным.

## 4. Инструкция по запуску
1. Установить Python 3.11(Обязательно, у меня на Python 3.14 не поставилась)  
2. Установить библиотеку Pygame:
   ```
   pip install pygame
   ```
3. Запустить файл с игрой:
   ```
   python main.py
   ```
## 5. Общие выводы
Pygame оказался удобным инструментом для изучения основ разработки игр. С его помощью можно быстро сделать простой проект и понять, как всё работает. Но для более сложных задач лучше использовать другие решения.
